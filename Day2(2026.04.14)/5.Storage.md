# 5. Storage

## volume
논리적으로 vm과 연결되어 영구 스토리지를 생성함
block storage 나 nfs형태로 구현됨

block storage: 대표적으로 SAN같은 것들이 있음 SAN스위치를 사용해서 FC로 연결함. 빠르게 데이터 주고받음 하지만 비쌈.
isci? 아이스카시는 싼데 느림.
분산 file system 구성하거나 DB구성하거나 vm의 영구 disk로서 스토리지백엔드 구성할 때 이걸 씀.
LVM은 공유 fs로는 쓰지말래.

## object storage
vm과 연결된 관걔가 아니다
비정형 형태로 데이터가 저장됨
